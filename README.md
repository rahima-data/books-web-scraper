# 📚 Books Web Scraper

My first ever Python project! A web scraper that collects book titles 
and prices from [books.toscrape.com](http://books.toscrape.com) — 
a website built specifically for scraping practice.

## 🔍 What It Does
- Visits books.toscrape.com automatically
- Scrapes all book titles and prices
- Saves the data to a CSV file

## 🔧 Libraries Used
- `requests` - fetches the webpage
- `BeautifulSoup` - parses the HTML

## 📊 Output
Saves a `books.csv` file containing:
- Book title
- Book price in £

## 🌟 What I Learned
- How to send HTTP requests with Python
- How to parse HTML with BeautifulSoup
- How to extract specific elements by tag and class
- How to save scraped data to a CSV file
- How to use Git and GitHub for the first time!
